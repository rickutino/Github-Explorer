## Getting started

* Clone this Repository in your terminal:
>    $ git clone https://github.com/rickutino/Github-Explorer.git
* Entry to directory:
>    $ cd Github-Explorer
* To install dependency:
>    $ yarn
* To run the project:
>    $ yarn start
